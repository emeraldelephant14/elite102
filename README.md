This is my Code2College banking project as part of elite102, and it is a simple Python application that interacts with a SQLite database. It serves as a demonstration of how to set up a Python project with SQLite and includes basic functionality for database operations.

The project includes a simple and clean CLI interface to manage a variety of functions, including:
- Create account
- Deposit
- Withdraw
- Check balance
- List accounts

Each number on the keyboard correlates to a function. Run the function to try it out!
